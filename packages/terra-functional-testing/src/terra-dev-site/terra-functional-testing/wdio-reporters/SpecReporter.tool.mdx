# Spec Reporter

The Spec Reporter writes test results to file in JSON format and is enabled by default in the `terra-functional-testing` wdio configuration.

## Configuration

Configuration options are defined using the `reporterOptions` key in the `wdio.conf.js`.

```js
// wdio.conf.js
const config = require('@cerner/terra-functional-testing/lib/config/wdio.conf').config;

config.reporterOptions = {
  // Reporter override options go here.
};

exports.config = config;
```

## Options

### outputDir

A path to the output directory the results files will be written.

Type: `string`

Required: `false`

Default: The default output directory is `/tests/wdio/reports`.

Example:

```js
// wdio.conf.js
const config = require('@cerner/terra-functional-testing/lib/config/wdio.conf').config;

config.reporterOptions = {
  outputDir: '/reports'
};

exports.config = config;
```
