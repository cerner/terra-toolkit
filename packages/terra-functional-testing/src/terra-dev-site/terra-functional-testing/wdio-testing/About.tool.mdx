# WebDriverIO Testing

The terra-functional-testing library is a utility for developing tests using the [WebdriverIO Framework](https://webdriver.io/).

## Accessibility Testing

The testing library integrates [axe-core](https://github.com/dequelabs/axe-core) utilities for automating accessibility testing.

### Terra.validates.accessibility

The accessibility assertion is available globally. Running the assertion will run acessibility checks on the current document. If acessibility violations are found the the test step will fail. 

```js
it('should report no accessibility violations', () => {
  browser.url('/testing/route/');

  Terra.validates.accessibility(); // Fails if accessibility violations are found.
});
```

The accessibility assertion accepts [options](https://github.com/dequelabs/axe-core/blob/develop/doc/API.md#options-parameter) that will be passed to axe for the anylysis.

```js
it('should override a rule configuration', () => {
  browser.url('/testing/route/');

  // Rule override for this specific test.
  const options = { rules: { 'color-contrast': { enabled: false } } };

  Terra.validates.accessibility(options);
});
```
